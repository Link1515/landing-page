## How to Build / 怎麼把它 Build 起來

```
$ npm install
$ npm start
```

然後 Server 就會自動聆聽 port 4000 了，這時候您可以連接 http://localhost:4000 來進行開發


## How to Deploy / 怎麼發佈更新
```
$ npm run build
$ npm start
$ ./deploy
```

就會自動將statics push上gh-pages了
